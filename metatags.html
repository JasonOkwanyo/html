They tell the web browser information about the webpage

They are placed in the head tags

Some attributes used with meta are:
charset-for the character in coding we are using
name-for introducing other sub-attributes
content-for explaining sub-attributes
http-equiv="refresh" content="20"-for refreshing the page after the number of seconds in the content attribute
name="viewport" content="width=device-width,initial-scale=1.0 -for adjusting the view to the device type
